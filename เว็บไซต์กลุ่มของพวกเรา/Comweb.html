ระบบการจองซื้อตั๋วหนังอย่างง่าย

ที่มาและความสำคัญ

เนื่องจากกลุ่มของพวกเราได้มาประชุมกันเมื่อตอนเลือกหัวข้อโครงงานว่ากลุ่มของเราจะเลือกหัวขข้ออะไรดี ที่สามารถนำไปพัฒนาต่อ และเป็นประโยชน์ต่อคนที่มาใช้งาน เราจึงพูดกันถึงสิ่งที่ชอบ ทำให้ได้ข้อสรุปคือการทำระบบจองตั๋วหนัง ซึ่งปัญหาก็กลุ่มของพวกเรานั้นไม่ค่อยมีความชำนาญด้านการเขียนโปรแกรม เราจึงสร้างเป็นระบบการจองตั๋วแบบอย่างง่ายขึ้นมาแทน ซึ่งอาจจะนำไปพัฒนาจนเป็นระบบที่ใหญ่ได้ และใช้การได้จริงในอนาคต โครงงานนี้เป็นการศึกษาการจองตั๋วอย่างง่ายโดยผ่านการเขียนโปรแกรมภาษาซี ซึ่งจัดทำขึ้นเพื่อศึกษาการเขียนโปรแกรมภาษาซี และการสร้างสรรค์แอพพลิเคชั่นที่อาจจะสามารถนำไปใช้พัฒนาต่อได้ หากเกิดข้อผิดพลาดประการใด ผู้จัดทำขออภัยมา ณ ที่นี้ด้วย

จุดประสงค์ของโครงงาน

โครงงานในครั้งนี้มีจุดประสงค์เพื่อศึกษาพื้นฐานภาษา C โดยการเขียนโค้ดเกี่ยวกับระบบจองตั๋วหนังอย่างง่ายขึ้นมา โดยภายในนั้นมีระบบต่างประกอบไปด้วย การแก้ไขราคา การยกเลิกตั๋วหนังซึ่งมีการใช้ Function พื้นฐานต่างๆ ประกอบไปด้วย If.....else, switch case, structure, system มาใช้ในการเขียน

รายละเอียดเกี่ยวกับโครงงาน

โครงงานนี้ที่เราทําขึ้นมาจะเป็นเหมือนโปรแกรมเล็กๆ ที่ใช้ ในการจองที่นั่งในโรงหนังและนอกจากการจองตั๋วหนังเราก็ยังมีระบบอื่นๆ อีกอย่างเช่นการตรวจสอบการจองตั๋ว การจองตั๋ว การยกเลิกการจองตั๋ว และออกจากระบบจอง โดยวิธีการจองตั๋วหนังนั้นทำได้ง่ายดังนี้ ขั้นแรกเราทำการ เลือกตัวเลือกการจองหนังจากนั้นเราทําการเลือกหนังเรื่อง ที่เราต้องการที่จะดู จากนั้นก็เลือกที่นั่งให้ชื่อเบอร์โทรศัพท์ จากนั้นก็เป็นอันเสร็จสิ้นการจอง

จองตั๋วได้ง่ายๆ ทำอย่างไรเรามาดูกัน

ก่อนอื่นเมื่อเรารันโปรแกรมขึ้นมาแล้ว จะมีตัวเลือกขึ้นมาให้เลือก 4 ตัวเลือกนั่นคือ
1.การตรวจสอบการจองตั๋ว
2.การจองตั๋วหนัง
3.การยกเลิกการจองตั๋ว
4.ออกจากระบบ เมื่อทำการเลือกตัวเลือกที่ 2 จะพาไปสู่หน้าการเลือกหนังที่คุณจะรับชม โดยหนังที่ฉายอยู่ตอนนี้ประกอบไปด้วย
1.Jujutsu kaisen : Zero
2.Lightyear
3.Detective Conan The Movie 25 เมื่อทำการเลือกเสร็จแล้วจะนำมาสู่หน้าการเลือกที่นั่ง ซึ่งมีที่นั่งทั้งหมด 56 ที่นั่ง และต้องกรอกรายละเอียดประกอบด้วย ชื่อ เบอร์โทรศัพท์ของคุณ และที่นั่งที่คุณต้องการนั่ง หลังจากนั้นการจองตั๋วของคุณก็เสร็จเรียบร้อย และจะมีการโชว์รายละเอียดต่างๆของการจองตามรูปด้านล่าง ต่อมาจะเป็นการตรวจสอบการจองหนัง เมื่อเลือกแล้วจะมีการโชว์รายละเอียดของตั๋วที่ทำการจองไปแล้ว เป็นอันเสร็จขั้นตอน

ถ้าหากว่าอยากจะยกเลิกการจองล่ะ

ขั้นแรกก็กดเลข 3 ที่ main menu กันเลย จากนั้นระบบก็จะขึ้นให้เลือกหนังที่เราต้องการยกเลิก เราก็กดเลขตามหนังที่เราได้ทำการจองไปตอนแรก และทำการกรอกเลข ID ของตั๋ว ให้ถูกต้อง ซึ่งถ้าหากกรอกผิดระบบจะกลับไปที่หน้า main menu เมื่อกรอกเสร็จแล้วระบบก็จะขึ้นประกาศชื่อว่าทำการยกเลิกการจองแล้ว เป็นอันเสร็จขั้นตอน
